How to Implement a Registration Form

This article has been removed because it only explained things that are already explained in other articles. Specifically, to implement a registration form you must:

  1. Define a class to represent users;

  2. Create a form to ask for the registration information (you can generate this with the make:registration-form command provided by the MakerBundle [https://symfony.com/doc/current/bundles/SymfonyMakerBundle/index.html]);

  3. Create a controller to process the form;

  4. Protect some parts of your application so only registered users can access to them.
